### Featured Projects

#### [Javino - AI Media Authenticity Engine](https://github.com/SriramGandhiS/Javino-AI-Authenticity)
An enterprise-grade forensic analysis platform designed to authenticate media in real-time. It combines EXIF metadata forensics, vision model inspection, and custom LLM evaluation to detect deepfakes.
* **Tech**: Spring Boot, Next.js, FastAPI, MySQL, Python, Groq API

#### [HireSense](https://github.com/SriramGandhiS/Hiresense.ai) – AI-Powered Mock Interview Platform
A full-stack placement prep application designed for resume analysis, keyword extraction, and proctored mock interviews with real-time speech-to-text transcription.
* **Tech**: React, Node.js, Express, MongoDB, Groq API, Chart.js

#### [Rules of India (ROI)](https://github.com/SriramGandhiS/ROI-THE-LEGAL-APP) – Legal Learning Platform
A cross-platform mobile and web legal assistant designed to democratize legal awareness for Indian citizens, featuring multilingual chats and comparative transitional law maps.
* **Tech**: Flutter, React, Firebase, Groq Llama 3, Gemini Pro

#### [SmartSlate](https://github.com/SriramGandhiS/SmartSlate) – Face Recognition Attendance Ecosystem
An AI-driven biometric attendance system utilizing OpenCV and LBPH face recognition for sub-second, frictionless edge logging synced to a global cloud database.
* **Tech**: Python, Flask, OpenCV, SQLite, MongoDB, Groq LLM

#### [CSE UTSAV 2K26](https://github.com/SriramGandhiS/UTSAV2K26) – Event Management Platform
A high-traffic serverless event management system designed to handle registration workflows for over 1,500 students with 99.9% uptime.
* **Tech**: HTML5, CSS3, JavaScript, Google Apps Script
